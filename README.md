# 项目使用方法

## 1. 安装依赖
```shell
git clone https://github.com/Guxianfu/uniapp-vue3-ts-vite-pinia.git
yarn install
<!--如果vue报错,可以尝试安装 -->
yarn upgrade
```

## 2. 运行项目
```shell
yarn run dev:h5
```

## 3. 打包项目
```shell
<!--最好是用hbuildx打包 -->
yarn run build:h5
```
